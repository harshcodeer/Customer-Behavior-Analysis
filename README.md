# Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow, covering data extraction, cleaning, analysis, visualization, and reporting. The objective is to transform raw data into meaningful insights using Python, SQL Server, and Power BI, and present the findings through a professional report and presentation.

---

## Dataset

The dataset contains business-related records used for analysis and visualization. It includes multiple attributes such as customer information, product details, sales transactions, and performance metrics.

### Key Features

* Structured tabular data
* Missing and duplicate records handling
* Suitable for exploratory and business intelligence analysis

## Project Workflow

### 1. Data Loading

* Imported dataset into Python.
* Examined data structure and data types.
* Verified dataset integrity.

### 2. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Corrected inconsistent data formats.
* Prepared data for analysis.

### 3. Exploratory Data Analysis (EDA)

* Generated descriptive statistics.
* Analyzed distributions and trends.
* Identified patterns and correlations.
* Created visualizations for better understanding.

### 4. SQL Analysis

* Imported cleaned data into SQL Server.
* Wrote SQL queries to answer business questions.
* Performed aggregations, filtering, and joins.
* Extracted key performance metrics.

### 5. Power BI Dashboard Development

* Connected Power BI to processed data.
* Designed interactive dashboards.
* Created KPIs, charts, filters, and slicers.
* Enabled business performance monitoring.

### 6. Reporting

* Documented project objectives and methodology.
* Summarized analytical findings.
* Included visualizations and business insights.

### 7. Presentation Creation

* Prepared professional presentation using Gamma.
* Highlighted project workflow, findings, and recommendations.
* Created stakeholder-friendly slides.

## How to Run

### Prerequisites

* Python 3.x
* SQL Server
* Power BI Desktop

### Steps

1. Clone the repository.

```bash
git clone <repository-link>
```

2. Install required Python libraries.

```bash
pip install pandas numpy matplotlib seaborn
```

3. Open and execute the Python notebooks.

4. Import the cleaned dataset into SQL Server.

5. Run the SQL queries available in the SQL folder.

6. Open the Power BI dashboard file (`.pbix`) and refresh the data.

7. Review the project report and presentation for detailed insights.
